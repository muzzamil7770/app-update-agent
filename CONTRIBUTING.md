# Contributing

## Setup

```bash
git clone https://github.com/muzzamil7770/app-update-agent
cd app-update-agent
npm install
npm run build
```

## Making Changes

1. Fork the repo and create a branch from `main`
2. Make your changes in `src/`
3. Run `npm run build` to verify it compiles
4. Open a pull request

## Commit Style

Use conventional commits:

```
feat: add retry config option
fix: prevent duplicate prompts on tab focus
chore: bump deps
```

## Reporting Issues

Use the GitHub issue templates — bug reports and feature requests are both welcome.
